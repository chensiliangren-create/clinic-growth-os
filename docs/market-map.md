# 整骨院業界の市場構造マップ

**参照:** [docs/project-charter.md](project-charter.md) - ターゲット、判断基準

業界全体の課題構造を理解し、どこで価値を提供するかを明確にします。

---

## 1. 集客（New Patient Acquisition）

### 現在の主流手段
- **Google検索 + Googleビジネスプロフィール**
  - 低コスト
  - 新規患者の入り口として必須
  - 口コミが評価に直結

- **紹介（口コミ）**
  - 既存患者の家族・知人
  - 医師・他業種からの紹介
  - 信頼度が高い

- **SNS（Instagram、Facebook、LINE）**
  - 若年層への露出
  - 定期的な更新が負担
  - 効果測定が曖昧

- **ポータルサイト（EPARK、ホットペッパービューティー等）**
  - 掲載費・手数料が高い
  - 利用者が限定的な場合も多い

- **自院ホームページ・ブログ**
  - 制作費がかかる
  - 更新頻度が課題

### 院長の悩み
- 「Google検索での上位表示が難しい」
- 「ポータルの広告費対効果が不明」
- 「SNSの更新が続かない」
- 「新規患者が急に減ることがある」
- 「地元で認知されていない」

### 市場ニーズ
- **Googleビジネス最適化** → 低コスト・検索流入増加
- **口コミ獲得の仕組み** → 信頼度向上
- **低予算・自運用可能な広告** → リスク低減
- **新規数の可視化** → 効果測定

---

## 2. リピート（Patient Retention & Frequency）

### 現在の主流手段
- **予約リマインド（電話・SMS・LINE）**
  - 手動が多い
  - 時間がかかる

- **LINE公式アカウント**
  - 導入は進んでいるが、運用が未熟
  - クーポン・キャンペーン活用が限定的

- **回数券・継続プラン**
  - 設計が難しい
  - 院長による試行錯誤

- **患者教育（ニュースレター・セミナー）**
  - 実施している院は少ない
  - 継続性が課題

- **新患フォロー**
  - 初回来院後の追跡が弱い
  - 2回目来院への施策がない

### 院長の悩み
- 「初回患者が戻ってこない」
- 「LINE登録率が低い」
- 「リマインド送信が手作業」
- 「リピート率が30%以下」
- 「既存患者との接触機会がない」

### 市場ニーズ
- **自動リマインド仕組み** → 来院率向上、手間削減
- **LINE運用テンプレ** → 継続的な接触
- **回数券・定期プラン設計支援** → 売上安定化
- **初回フォロー導線** → リピート率向上
- **患者離脱防止プログラム** → リピート率可視化

---

## 3. 求人・スタッフ（Recruitment & Staff Management）

### 現在の課題
- **採用難**
  - 給与が低い傾向
  - 労働条件が悪い認識
  - 小規模院は知名度がない

- **定着率の低さ**
  - 業務が多岐にわたる
  - キャリアパスが不明確
  - 教育体制が整っていない

- **業務負担**
  - 施術者が受付も兼務
  - 経営事務も院長が
  - スタッフ教育の余裕がない

### 市場ニーズ
- **求人・採用支援** → 採用コスト削減
- **スタッフ育成プログラム** → 定着率向上
- **業務効率化** → スタッフ負担低減
- **経営事務の自動化** → 院長の時間確保

### Clinic Growthとの関連性
- **低・中優先度**（初期は集客・リピートに注力）
- ただし、スタッフ負担を減らす施策は、院長の施術集中を実現する手段

---

## 4. 教育・スキル（Professional Development）

### 現在の課題
- **継続教育の機会が限定的**
  - 技術研修は有料で高額
  - 経営・マーケティング学習がない
  - 情報源が広がらない

- **新しい知識へのアクセス**
  - SNS運用、LINE活用、Googleビジネスの最新情報
  - ノウハウが古いまま

- **コミュニティの不在**
  - 同業他社との情報共有が限定的
  - 孤立しやすい

### 市場ニーズ
- **経営・集客セミナー** → 実務知識習得
- **デジタルツール活用研修** → スキルアップ
- **院長向けコミュニティ** → 情報交換

### Clinic Growthとの関連性
- **初期は低優先度**（ノウハウ提供は我々の本来事業ではない）
- 相互紹介・提携による補完は検討可能

---

## 5. 経営管理（Business Management & Analytics）

### 現在の課題
- **数字の可視化が不十分**
  - 売上・経費は把握しているが、詳細分析がない
  - 患者LTV（生涯価値）を知らない
  - ROI（広告効果）が不明
  - 月次の利益予測ができない

- **経営意思決定の根拠が曖昧**
  - 「感覚」で判断している
  - 失敗と成功の分析がない
  - 次のアクションを決めづらい

- **税務・会計処理**
  - 税理士との関係が限定的
  - 節税対策の知識がない
  - 経営計画書がない

### 院長の悩み
- 「本当に利益が出ているのか分からない」
- 「集客費をいくら使うべきか分からない」
- 「今月の利益予測ができない」
- 「誰に相談すれば良いか分からない」
- 「経営指標って何が必要？」

### 市場ニーズ
- **簡易KPIダッシュボード** → 経営可視化
- **月次経営分析** → 意思決定支援
- **ROI計測・分析** → 施策判断基準
- **利益予測ツール** → キャッシュフロー管理
- **経営顧問サービス** → 顧問機能

### Clinic Growthとの関連性
- **高・中優先度**（集客・リピート成果を経営数字へ統合）
- 経営改善は院長の安心感と信頼を生む

## 市場全体構造図

```
整骨院・治療院の経営課題

┌─ 集客（新規患者）
│   ├─ Google検索
│   ├─ 口コミ・紹介
│   └─ SNS・広告
│
├─ リピート（継続来院）
│   ├─ 予約管理
│   ├─ リマインド
│   └─ 継続施策
│
├─ 経営管理（利益・成長）
│   ├─ 売上分析
│   ├─ 費用管理
│   └─ 利益予測
│
├─ スタッフ（組織・人材）
│   ├─ 採用・定着
│   └─ 教育・育成
│
└─ 教育（知識・スキル）
    ├─ 技術研修
    ├─ 経営学習
    └─ 業界情報
```

## Clinic Growthの介入ポイント

### 初期フェーズ（1院支援）
- 集客（Google・口コミ）
- リピート（LINE・リマインド）
- 経営管理（簡易KPI）

### 中期フェーズ（3～10院）
- 上記を深化
- 初期採用・スタッフ効率化

### 長期フェーズ（30院以上・OS化）
- すべての領域を統合
- 業界スタンダード化
