# 共通課題仮説集

**参照:** [docs/project-charter.md](project-charter.md) - ターゲット、成長戦略（フェーズ1）

以下は現時点の仮説です。30人インタビューで検証し、事実を基に更新します。

判断基準: 「院長の実務課題であること。スタッフ1～5名の小規模院に実際に起きている課題であること。"

1. 新規集客の不足
- 検索流入・広告・口コミが弱い可能性がある。

2. リピート率の低さ
- 初回は来るが継続につながりにくい可能性がある（効果の伝達不足、フォロー不足）。

3. 予約の空きとキャンセル
- 平日昼や特定時間帯に穴がある可能性がある。キャンセル対策が不十分な可能性。

4. 単価・保険収入への依存
- 保険収入に依存して利益が薄い可能性がある。保険外サービスを十分に売れていない可能性。

5. スタッフの採用・定着
- 小規模院で教育や評価が難しい可能性がある。人手不足が運営を圧迫している可能性。

6. 経営データの可視化不足
- 売上・LTV・CPAなどの指標が見えず、打ち手が曖昧になっている可能性がある。

次のステップ:
- 30人の院長ヒアリングで仮説を検証し、共通の事実を見つける
- 事実をもとに優先課題と初期MVPを決める
