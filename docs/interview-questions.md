# インタビュー用質問（問題を決めつけない質問）

**参照:** [docs/project-charter.md](project-charter.md) - 判断基準の「市場調査・インタビューの確認リスト」

この質問は、院長の現場感を聞き取り、事実と感情を整理するためのものです。

判断基準: 「開かれた質問を使う。『～ですか？』ではなく『どのようなことですか？』『なぜだと思いますか？』"

1. 最近、院の運営で一番気になっていることは何ですか？
2. ここ数ヶ月で「うまくいった」と感じたことはありますか？それは何ですか？
3. 「もっと良くしたい」と思っている業務やプロセスはどこですか？
4. 今の患者さんの来院理由や反応で印象に残っていることはありますか？
5. 予約・集客・問合せの流れで課題だと感じる場面はありますか？
6. スタッフや現場の負担で悩んでいることはありますか？
7. 現在、外部の支援やツールで使っているものはありますか？その満足度は？
8. これから半年以内にどんな変化があればいいと思いますか？

簡単なスクリーニング:
- 施術スタッフの人数
- 主要な顧客層
- 1日あたりの患者数の目安
