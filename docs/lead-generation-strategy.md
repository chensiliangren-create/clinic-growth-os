# 見込み客発見戦略 - Google Maps API活用

**参照:** [docs/project-charter.md](project-charter.md) - 判断基準

Google Maps / Places APIを活用して、Web集客導線が弱い整骨院を特定し、無料ヒアリングへ誘導する営業戦略です。

---

## 基本戦略

### ビジネスモデル

```
Google Maps / Places API で公開情報を取得
         ↓
Web集客導線の強度をスコアリング
         ↓
見込み度の高い院をランク付け
         ↓
院ごとに簡易診断レポート自動生成
         ↓
「今、こんな改善機会があります」と提案
         ↓
無料ヒアリング申し込み
         ↓
実行支援へ→Clinic Growth OS化
```

### 利用API

**Google Maps Platform / Places API**
- 検索条件：「整骨院」「接骨院」「カイロプラクティック」
- 地域指定：ターゲット都市別
- 取得項目：
  - 基本情報（名前、住所、電話、Webサイト）
  - Google口コミ（数、平均評価）
  - 営業時間、写真、投稿
  - 予約ボタン有無

**規制事項**
- ✅ 公式API利用（スクレイピング不可）
- ✅ API利用規約遵守
- ✅ 1日のクエリ上限を確認（月額制）
- ✅ 取得データは営業用途で保存・利用可能（規約確認）
- ✅ 個人情報（評価者情報）は取得・保存不可
- ⚠️ 電話番号取得について：公式情報のみ
- ⚠️ 大量取得時には事前に Google に相談

## 見込み客の対象条件

### 1. 業種
- 整骨院（Chiropractor, Orthopedic Clinic）
- 接骨院（Bonesetter）
- 鍼灸院・マッサージ（Acupuncture, Massage）
- 治療院全般

### 2. 立地条件
- 都市部：人口 50万人以上の市（新規患者数見込み）
- 郊外：駅から徒歩 15分以内
- 新規開業予定地：ビル完成前後

### 3. Web集客導線の弱さ（スコアリング対象）

| 項目 | 弱い状態（対象） | スコア |
|------|---------|--------|
| **Webサイト** | なし、または Google へのリンクがない | -3 |
| **Google口コミ** | 0～5件 | -2 |
| **Google口コミ** | 6～15件 | -1 |
| **口コミ評価** | 3.5未満 | -1 |
| **写真数** | 0～5枚 | -1 |
| **投稿頻度** | 3ヶ月以上更新なし | -2 |
| **予約ボタン** | Google 予約ボタンなし | -2 |
| **予約導線** | 予約システムが見当たらない | -2 |
| **LINE導線** | Google 上に LINE リンクなし | -1 |

**スコアリング基準**
- スコア -12～-8：最優先対象（弱い院）
- スコア -7～-4：高優先度（改善機会大）
- スコア -3～0：中優先度（成長段階）

## 見込み客スコアリング詳細

### 各指標の定義と優先度

#### 1. Webサイト有無（優先度：⭐⭐⭐⭐⭐）
- **基準**
  - ✓ 独立したホームページがある
  - ✓ Google ビジネスプロフィール欄に「ウェブサイト」リンクあり
  - ✗ ホームページなし
  - ✗ Wix、Ameba 等の無料テンプレのみ

- **判断方法**
  - API: `website` フィールドが存在して URL を返すか
  - 手動確認：その URL にアクセスして、「院の情報」が書かれているか

- **スコア影響度**
  - Webサイトなし：-3（集客導線が完全に弱い）
  - 弱いサイト：-1

#### 2. Google口コミ数（優先度：⭐⭐⭐⭐⭐）
- **基準**
  - 0～5件：見込み客を獲得できていない
  - 6～15件：初期段階だが改善の余地あり
  - 16～30件：ある程度回っているが、改善余地あり
  - 31件以上：充実している

- **判断方法**
  - API: `review_count` フィールドの値

- **スコア影響度**
  - 0～5件：-2（最優先）
  - 6～15件：-1（改善機会あり）

#### 3. 口コミ評価（優先度：⭐⭐⭐）
- **基準**
  - 3.5未満：課題がある可能性
  - 3.5～4.2：平均的
  - 4.3以上：高評価

- **判断方法**
  - API: `rating` フィールドの値

- **スコア影響度**
  - 3.5未満：-1（改善機会あり）

#### 4. 写真数（優先度：⭐⭐⭐）
- **基準**
  - 0～5枚：施設情報が不十分
  - 6～15枚：基本的には揃っている
  - 16枚以上：積極的に運用

- **判断方法**
  - API: `photos` 配列のカウント

- **スコア影響度**
  - 0～5枚：-1

#### 5. 投稿頻度（優先度：⭐⭐⭐⭐）
- **基準**
  - 3ヶ月以上更新なし：SNS/Google 投稿が放置
  - 1～3ヶ月：たまに更新
  - 月1回以上：継続的に運用

- **判断方法**
  - API: 各 `review` / `photo` の `time` フィールドから最新日付を抽出
  - 現在日付との差分を計算

- **スコア影響度**
  - 3ヶ月以上：-2（改善の余地大）

#### 6. 予約導線（優先度：⭐⭐⭐⭐⭐）
- **基準**
  - ✓ Google 予約ボタンあり
  - ✓ 外部予約システムへのリンクあり（ホットペッパー等）
  - ✗ 予約方法が見当たらない
  - ✗ 電話のみ（Webサイトなし）

- **判断方法**
  - API: `business_status` が "OPERATIONAL" か
  - `formatted_phone_number` のみで他に導線がないか
  - 手動確認：Google 上に予約ボタンがあるか

- **スコア影響度**
  - 予約導線なし：-2（改善機会大）

#### 7. LINE導線（優先度：⭐⭐⭐）
- **基準**
  - ✓ Webサイトに LINE リンクあり
  - ✓ Google 上に LINE リンク表示
  - ✗ LINE 導線が見えない

- **判断方法**
  - 手動確認：Webサイトに LINE リンクがあるか
  - API: `website` の URL にアクセスして LINE の QR や URL を探す

- **スコア影響度**
  - LINE 導線なし：-1

## 営業アプローチ

### アウトリーチ方法

**方法1: 直接メール/電話**
- 件名：「Google 検索で見つけた、 XXX 整骨院の Web 集客課題」
- 本文：簡易診断レポートを 3～5行で要約
- CTA：「無料ヒアリングにお申し込みください」

**方法2: Google ビジネスプロフィール経由**
- Google 上の「メッセージ」機能を使った直接連絡
- 短く、価値を伝える

**方法3: チラシ＋QR コード**
- 対象院の立地が判明したら、その地域にポスティング
- QR から無料ヒアリング LP へ

### アウトリーチの倫理的配慮

✅ **やること**
- 公開情報のみを使用
- 価値提案を先に（営業ではなく診断提案）
- 「今、改善機会があります」というトーン
- 無料ヒアリングという低リスク提案

❌ **やってはいけないこと**
- 院長の個人メールを無断取得
- 「ノウハウ販売」のような勧誘
- 高圧的な営業
- 複数回の営業アプローチ（1回限定）
- 価格話から入る

## 注意点と制限

### API制限
- 月額制の料金（Places API は従量制）
- 1 日のリクエスト数には制限あり
- 大規模取得時は Google への事前相談必須

### 保存可能項目
- ✓ 店名、住所、電話、Webサイト URL
- ✓ 口コミ数、評価、最新投稿日
- ✓ 営業時間、 photo 数、予約有無
- ✗ 個人情報（口コミ投稿者、メールアドレス）
- ✗ 口コミテキスト（著作権）

### 営業倫理
- 迷惑営業との線引き
  - ✓ 価値提案が先（診断レポート）
  - ✗ セールス圧力（複数接触、返信強要）
  - ✓ 相手の判断尊重（1 回の提案で決める）
