# 顧客ジャーニー - 院長の困り事の変化

**参照:** [docs/project-charter.md](project-charter.md) - ターゲット、判断基準

院長のビジネスステージ（開業前 → 開業直後 → 成長期 → 安定期）ごとに、困っていることと必要な支援が何かを整理します。

---

## ステージ1: 開業前（Pre-Launch）

### ビジネス状況
- 資金を集めている段階
- 物件探し、内装工事、機器購入を計画中
- スタッフ採用を検討
- 経営数字の予測（患者数、売上、利益）

### 主な困り事

| 困り事 | 院長の声 | 優先度 |
|--------|---------|--------|
| 集客戦略が分からない | 「どうやって新患を集めるのか」 | ⭐⭐⭐⭐⭐ |
| 初期投資の意思決定 | 「ホームページは必要か、予約システムは」 | ⭐⭐⭐⭐ |
| 経営計画の立て方 | 「最低限の患者数は何人」 | ⭐⭐⭐⭐ |
| 競合分析 | 「この地域に何院ある、どう差別化」 | ⭐⭐⭐ |
| スタッフ採用・育成 | 「誰を雇う、どう育てる」 | ⭐⭐⭐ |

### 市場での支援状況
- ほぼない（銀行、税理士が部分的にサポート）

### Clinic Growthの介入機会
- ❌ **まだ関係ない** - 患者がいない段階では支援できない
- 可能性：開業直前の「初期運用テンプレ」提供（低優先度）

### 次ステージへの判断基準
- 「開業資金が集まった」
- 「物件が決まった」
- 「1ヶ月以内に開業」

---

## ステージ2: 開業直後（Post-Launch, 0～6ヶ月）

### ビジネス状況
- 開業！
- 初患者がようやく来院
- 月10～50患者くらいの状況
- 月商は5～30万円くらい（変動大）

### 主な困り事

| 困り事 | 院長の声 | 優先度 |
|--------|---------|--------|
| **新患が来ない** | 「誰も知らない。何もしてないから」 | ⭐⭐⭐⭐⭐ |
| 予約・受付が手動 | 「受付スタッフの雇用が負担」 | ⭐⭐⭐⭐ |
| リピート率が低い | 「初回来たら、誰も戻ってこない」 | ⭐⭐⭐⭐⭐ |
| キャッシュフロー | 「赤字がこのまま続くのか」 | ⭐⭐⭐⭐⭐ |
| Googleビジネス何も分からない | 「何をしたらいい」 | ⭐⭐⭐⭐ |
| 周辺認知がない | 「この辺に治療院があること、誰も知らない」 | ⭐⭐⭐⭐ |

### 市場での支援状況
- ホームページ制作会社：「ホームページ作りませんか」（高額、費用対効果不明）
- ポータルサイト：「EPARK登録しませんか」（手数料が負担）
- SNS：「Instagramやってますか」（知見がない）

### Clinic Growthの介入機会
- ✅ **最適な段階**
- Google検索 + 口コミ戦略（集客）
- LINE登録 + リマインド（リピート）
- 簡易KPI管理（経営見える化）

### このステージで実現すべき成果
- 月30～50新患を安定的に獲得
- リピート率を30～50%に向上
- 月商を50～100万円に
- キャッシュフロー黒字化へ

### 次ステージへの判断基準
- 「月30新患が安定的に来ている」
- 「リピート率が30%以上」
- 「月商が100万円を超えた」

---

## ステージ3: 成長期（Growth, 6ヶ月～2年）

### ビジネス状況
- 毎月30～80新患
- 月商100～300万円
- スタッフ2～3名に増員
- 経営の課題が多角化

### 主な困り事

| 困り事 | 院長の声 | 優先度 |
|--------|---------|--------|
| スタッフの育成・定着 | 「スタッフが育たない、辞めてしまう」 | ⭐⭐⭐⭐⭐ |
| 集客コストの最適化 | 「広告費をいくら使うべき」 | ⭐⭐⭐⭐ |
| 利益率の低下 | 「売上は増えるが、利益が増えない」 | ⭐⭐⭐⭐⭐ |
| 予約・スケジュール管理 | 「スタッフのシフト管理が煩雑」 | ⭐⭐⭐ |
| 患者管理の複雑化 | 「どの患者が何の施術をいつ」か分からない | ⭐⭐⭐⭐ |
| 経営意思決定 | 「次は何に投資すべき」 | ⭐⭐⭐⭐ |
| 競合の出現 | 「近所に新しい院ができた」 | ⭐⭐⭐ |

### 市場での支援状況
- 経営コンサル：提案し始めるが、高額（月10～30万円）
- スタッフ研修会：「こんなセミナー受けませんか」
- 会計士：税務相談のみ

### Clinic Growthの介入機会
- ✅ **深化のフェーズ**
- 集客の質向上（ROI分析）
- リピート率50%超への施策
- 利益改善（受付効率化、スタッフ時短）
- 簡易経営分析（患者LTV、ROI、利益予測）

### このステージで実現すべき成果
- 月80～120新患
- リピート率50～60%
- 月商300～500万円
- 利益率20～30%
- スタッフ定着率向上

### 次ステージへの判断基準
- 「月商500万円を超えた」
- 「スタッフが安定している」
- 「経営が自動化されている」

---

## ステージ4: 安定期（Mature, 2年以上）

### ビジネス状況
- 毎月80～150新患（安定）
- 月商300～800万円
- スタッフ3～5名
- 経営が一定の安定
- 複数の収益源を検討

### 主な困り事

| 困り事 | 院長の声 | 優先度 |
|--------|---------|--------|
| さらなる成長への天井 | 「月商1000万円は可能か」 | ⭐⭐⭐ |
| 多店舗展開の検討 | 「2号店を出すべきか」 | ⭐⭐⭐ |
| 組織マネジメント | 「スタッフの自動化がしたい」 | ⭐⭐⭐⭐ |
| 新商品・新サービス開発 | 「新メニュー追加したい」 | ⭐⭐⭐ |
| 患者層の変化への対応 | 「高齢患者が増えてきた」 | ⭐⭐⭐ |
| 継承・後継者育成 | 「いつまで自分が現役でやるか」 | ⭐⭐ |
| プロフェッショナル化 | 「更に信頼度を高めたい」 | ⭐⭐ |

### 市場での支援状況
- 経営コンサル：複数から提案されている
- 業界団体：協会費用で情報提供
- 同業者ネットワーク：自然発生的に形成

### Clinic Growthの介入機会
- ✅ **プラットフォーム化のフェーズ**
- Clinic Growth OS の本格導入
- 多店舗管理ツール
- スタッフの行動自動化
- 経営AI分析
- 業界コミュニティへの昇格

### このステージで実現すべき成果
- 月商800万円～1000万円
- 利益率30%超
- スタッフ満足度向上
- 新規事業の検討段階へ

## ステージ別の対応戦略

### 開業前 → 開業直後
- **タイミング：** 開業1ヶ月前から接触
- **メッセージ：** 「最初の6ヶ月で月30新患を達成させます」
- **支援内容：** 初期テンプレ + Google設定サポート

### 開業直後（Clinic Growth 集中投資期）
- **タイミング：** 開業直後～6ヶ月
- **メッセージ：** 「新患獲得×リピート化で自走できる仕組みを作ります」
- **支援内容：** 人力伴走 + テンプレ + KPI管理 + 成果ベース

### 成長期
- **タイミング：** 月商100万円突破後
- **メッセージ：** 「利益改善と組織化で月商500万円へ」
- **支援内容：** 経営分析 + スタッフ効率化 + 次フェーズ検討

### 安定期
- **タイミング：** 月商500万円以上
- **メッセージ：** 「Clinic Growth OS で多店舗化・スケール」
- **支援内容：** プラットフォーム提供 + マッチング + 業界ネットワーク

---

## 顧客ジャーニーから見た Clinic Growth の価値

1. **タイミング** - 最も困っている「開業直後」に現れる
2. **問題解決** - 集客とリピートの「つなぎ」を実現
3. **段階性** - 成長に合わせて支援レベルを上げられる
4. **測定可能性** - 各ステージで明確な成功指標がある
5. **継続性** - 一度の支援で終わらず、成長期まで伴走可能
