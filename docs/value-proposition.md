## 中核的な価値提案

### Clinic Growth の本質的価値

```
┌──────────────────────────────────────────────────┐
│                                                  │
│ 「先生が気づいていないWeb集客上の                │
│  機会損失を見つけ、実行可能な改善策として       │
│  提示する」                                      │
│                                                  │
│ + 伴走支援で実現まで一緒に行う                  │
│                                                  │
└──────────────────────────────────────────────────┘
```

### 院長の現状

- 「新患が安定しない理由が分からない」
- 「何をしたら改善するのか、何から始めるべきか不明」
- 「Google検索やSNS対策をしているが、何が足りないか気づかない」
- 「Webサイトやリピート施策は後回しにしがち」

### 我々ができること

1. **機会損失の発見**
   - Google Maps 上の公開情報からWeb集客の弱点を可視化
   - 「口コミが3件 → 改善できれば月50%増加」といった具体的な改善見積もり
   - 院長が「気づかなかった改善機会」を提示

2. **診断と提案**
   - 院長のヒアリングを通じて、実務的な改善施策に落とし込む
   - 「今月はこれをやる、来月はこれ」という実行計画を提案

3. **伴走支援**
   - テンプレとチェックリストで、院長が実行しやすくする
   - 月1回の進捗面談で、改善の回転速度を上げる

## 基本的な価値提案（詳細）

### ターゲット
**30～50歳の治療院院長**
- 1院を自分たちで運営している
- スタッフは1～5名
- Google検索や口コミで新患を獲得している
- 経営数字を把握していないか、把握しようとしている

### 主な困り事
1. 新患が安定しない（何が原因か分からない）
2. 初回患者が戻ってこない（リピート率が低い）
3. 経営がどうなっているか分からない
4. 時間がなく、自分でやるしかない

### Clinic Growthの3段階の価値

```
段階1：「診断」← 営業ハードル低い、院長に価値を感じさせる
     ↓
段階2：「無料ヒアリング」← 具体的な改善策を提案
     ↓
段階3：「実行支援」← テンプレ + 伴走で実現化
```

## 提供する価値（5つの層）

### Layer 0: Web集客上の機会損失の見える化（NEW）

**What we provide（何を）**
- Google Maps 公開情報から、Web集客上の弱点を自動診断
- 院ごとの改善機会を「スコア」と「具体的な改善見積もり」で提示

**How（どうやって）**
1. Google Maps / Places API から公開情報を取得
   - Webサイト有無
   - Google口コミ数・評価
   - 投稿頻度
   - 写真数
   - 予約導線・LINE導線

2. スコアリング（-12～0の範囲）
   - Webサイトなし：-3
   - 口コミ0～5件：-2
   - 投稿3ヶ月以上なし：-2
   - 予約導線なし：-2
   など

3. 簡易診断レポート自動生成
   - 「Google口コミが3件 → 15件に増えれば、新患が月50%増える可能性」
   - 「Webサイト導入 → Google検索からの流入が3倍に」
   - 「LINE導線確保 → リピート率が20%向上」

**Outcome（結果）**
- **院長の気づき**：「こんな改善機会があったのか」
- **営業へのハードル低下**：営業ではなく「診断提案」
- **ヒアリング成約率向上**：具体的な改善見積もりで説得力UP

**Compare to alternatives（対案比較）**
| 代案 | 方法 | 院長の反応 | 次のステップ |
|------|------|----------|------------|
| 何もしない | - | 接触なし | - |
| 営業電話 | 「サービス買いませんか」 | 警戒・拒否 | 進みにくい |
| **Clinic Growth** | 「Web集客診断です」 | 興味・好奇心 | **ヒアリングへ** |

### Layer 1: 集客の実現（New Patient Acquisition）

**What we provide（何を）**
- Google検索で上位に表示される仕組み
- 口コミを増やす仕組み
- 新患がちゃんと予約する流れ

**How（どうやって）**
1. Google ビジネスプロフィール最適化
   - 写真・動画の撮影
   - 説明文の改善
   - 営業時間の正確性チェック

2. 口コミ獲得フロー
   - 来院患者への口コミ依頼タイミング
   - 低評価への返信テンプレ

- **初期状態：** 月5～10新患
- **3ヶ月後：** 月20～30新患

   - 「ホームページ制作（¥50万）+ 月額（¥5万）vs. Clinic Growth（¥5万）」

   - ビデオテスティモニアル
   - ブログでの実例紹介
