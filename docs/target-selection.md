# ターゲット選定

優先ターゲット（調査対象）:
- プロフィール: 個人〜小規模院、施術1〜3名、地域密着型の院長
- 状態: 新規開業・安定経営・事業承継準備など多様なステージ
- 目的: まずは実際の声を広く拾い、共通課題と例外を分ける

具体的な候補ペルソナ:
1. 若手開業院長（#8／#24） — 事業の初期段階で現状を知りたい
2. 地方家族経営院（#3／#6） — 人手と集客の両方を抱える
3. 都内専門院（#4／#11／#29） — 競合と高単価のバランスを考える
4. 女性院長・美容系（#2／#10／#22） — 顧客体験と紹介が鍵
5. 予約アプリ/デジタル活用院（#26／#28） — ツール利用の評価を聞く

調査の視点:
- さまざまな院長の「実際の悩み」と「日常の業務」を聞く
- 課題の大小ではなく、頻度と再現性を重視する
- 現場の声から、次のMVP設計に使える共通テーマを抽出する
