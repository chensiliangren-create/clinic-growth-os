# 30人インタビュー計画

**参照:** [docs/project-charter.md](project-charter.md) - ターゲット、成長戦略（フェーズ1）

目的: 30人の院長への市場調査を通じて、共通課題と事実に基づく仮説を発見する。

判断基準: 「多様性を保つ（地域、規模、課題、スタッフ数）。各院の言葉で課題を記録。"

手順:
1. ターゲットリスト作成（docs/personas.mdを参考に、多様な院長30名を選定）
2. 1週間で30件のヒアリングを目指す（1日5件×6日など）
3. 市場調査アプローチに沿って連絡し、5〜10分のインタビューを実施
4. 各院の声を記録し、共通テーマと個別差異を整理
5. 30件完了後に仮説をまとめ、次のMVP構想に反映する

成果物:
- 30人分のヒアリングメモ
- 共通課題のマッピング
- 優先度の高い1〜2つの課題仮説
- 初期無料導入候補院1件の選定

準備物:
- `docs/market-research-approach.md` を使って連絡
- `docs/interview-questions.md` を元に聞き取り
- `docs/personas.md` で多様な院長像を把握

注意点:
- 本調査は売り込みではなく、課題の事実を集めること
- 最初は1院無料導入して事例化を目指す
- 長期は「人＋AI＋テンプレ→半自動化→OS化」の順で進める
